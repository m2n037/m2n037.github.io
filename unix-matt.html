Delve in Unix

A step-by-step guide to learn Unix from Matt Might's blog

This page is linked from Stuff.

This is how I learnt dealing with Ubuntu linux and shell scripting. I started with dual-booting ubuntu with Windows 8.1. Then using the blog posts of Matt Might I delved into advanced unix stuff.

The Sequential Guide to Learning Unix

  • Start with Survival guide for Unix newbies

  • Then polish it with Settling into Unix

  • Then, learn some bash with Shell programming with bash: by example, by counter-example

  • Go deep into bash using Wooledge Bash Guide and Advanced Bash Scripting Guide

  • Go deep further with the books

    • Unix Power Tools

    • UNIX and Linux System Administration Handbook

    • Classic Shell Scripting

  • Or, understand and tinker with a fun project in bash to improve your writing - 3 shell scripts to improve your writing, or “My Ph.D. advisor rewrote himself in bash.”

  • Find further resources of Linux/Unix from Tips, tricks and tools for Linux/Unix
