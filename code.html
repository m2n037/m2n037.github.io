How to Use in Windows

  • Download the repository by clicking Download Zip in the lower right corner and unzip it.

  • Install Python 3.x and create a shortcut for python.exe i.e. C:/Python32/python.exe in the folder where isz-tool.py is. This is to smoothly run isz-tool incase you use Python 2.x for your other needs. The shortcut will ensure that isz-tool.py runs with Python 3.x instead of default Python distribution you're using.

  • Copy isz-tool.py and the shortcut mentioned above to the folder where your ISZ file resided, and open command line in the folder by using Shift+RightClick and selecting Open Command Window here

  • run the commands as per the readme.md file for verifying or converting.
